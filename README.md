## Overview

The *specthingsdoc* repository provides documentation sources for the following
Python packages:

- [specitems](https://github.com/specthings/specitems)

- [specware](https://github.com/specthings/specware)

- [specmake](https://github.com/specthings/specmake)

- [spectestrunner](https://github.com/specthings/spectestrunner)

The repository is maintained by the
[specthings](https://github.com/specthings)
project.
